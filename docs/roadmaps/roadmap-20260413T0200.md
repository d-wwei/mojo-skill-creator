# Iteration Roadmap v2

## 当前状态速览

v2 已交付。新增 distribution-first 原则（#1）+ 可选 se-kit 集成。SKILL.md 475 词，7 红线，6 设计原则，8 reference 文件。两轮 review 通过。

## 本次迭代经验

### 做对了
- 用户提出"分发视角"后立即提升为设计原则，而非当作单点修复
- se-kit 集成采用 bundle 模式，终端用户零配置
- Review 抓到了 phase 排序和原则编号不一致的问题

### 需改进
- 第一次提交时跳过了 review → ship → compound 流程，被用户指出后补上。**教训：增量变更也必须走完 pipeline，不能因为"只是加了几个段落"就跳过**
- token 预估再次出现偏差（v1 review 已修过同类问题），说明每次增删内容后都应重新 `wc -w`

### 新发现
- "Distribution-first" 作为设计原则的效力远超 "self-contained" 作为技术要求 — 它改变了所有决策的默认视角
- "Fresh machine test" 是一个好的机械检验：下载到干净机器上，只装 agent，能不能直接用？
